# Text

Text is the used to render text and paragraphs within an interface. It renders a
`<p>` tag by default.

## Import

```js
import { Text } from "@chakra-ui/core"
```

## Changing the font size

To increase the font size of the text, you can pass the `fontSize` prop.

```jsx
<Stack spacing={3}>
  <Text fontSize="6xl">In love with React & Next</Text>
  <Text fontSize="5xl">In love with React & Next</Text>
  <Text fontSize="4xl">In love with React & Next</Text>
  <Text fontSize="3xl">In love with React & Next</Text>
  <Text fontSize="2xl">In love with React & Next</Text>
  <Text fontSize="xl">In love with React & Next</Text>
  <Text fontSize="lg">In love with React & Next</Text>
  <Text fontSize="md">In love with React & Next</Text>
  <Text fontSize="sm">In love with React & Next</Text>
  <Text fontSize="xs">In love with React & Next</Text>
</Stack>
```

### Truncate text

Pass the `isTruncated` prop to render an ellipsis when the text exceeds the
width of the viewport or `maxWidth` set.

```jsx
<Text color="gray.500" isTruncated>
  Lorem ipsum is placeholder text commonly used in the graphic, print, and
  publishing industries for previewing layouts and visual mockups.
</Text>
```

### Override style

You can change the entire style of the text via props. For example, to change
the font size, pass the `fontSize` prop. No need to write `css` or `styled()`

```jsx
<Text fontSize="50px" color="tomato">
  I'm using a custom font-size value for this text
</Text>
```

### Override the element

To override the element that gets rendered, pass the `as` prop. Use **Inspect
Element** to see the element that gets rendered in html.

```jsx
<>
  <Text as="i">Italic</Text>
  <br />
  <Text as="u">Underline</Text>
  <br />
  <Text as="abbr">I18N</Text>
  <br />
  <Text as="cite">Citation</Text>
  <br />
  <Text as="del">Deleted</Text>
  <br />
  <Text as="em">Emphasis</Text>
  <br />
  <Text as="ins">Inserted</Text>
  <br />
  <Text as="kbd">Ctrl + C</Text>
  <br />
  <Text as="mark">Highlighted</Text>
  <br />
  <Text as="s">Strikethrough</Text>
  <br />
  <Text as="samp">Sample</Text>
  <br />
  <Text as="sub">sub</Text>
  <br />
  <Text as="sup">sup</Text>
</>
```

## Baseline Crop

`Text` and `Heading` elements have a prop called `baselineCrop`. This prop
allows us to remove the extra white space that normally appears above and below
text.

> Note: This feature requires some manual setup in your theme file in order to
> function. See `One-time Setup` section below for details.

### One-Time Setup

In order to use `baselineCrop`, you must measure your font manually and enter
those measurements into the `baselineCrop` key in your theme.

How to measure your font(s):

1. Navigate to this website: <http://text-crop.eightshapes.com/>
2. Select your font in the dropdown or use `Custom Font` option.
3. Choose size `48` and line-height `1.2`.
4. Move the top/bottom crop lines until they line up with the `Cap Height` and
   `Baseline` of your font.
5. Insert the values into your theme

```js
const myTheme = {
  //...
  baselineCrop: {
    // replace `body` with key of the font in your theme's `fonts` object that you measured
    body: {
      topCrop: 10, // use the value you measured above
      bottomCrop: 9, // use the value you measured above
    },
    // Other fonts go here
  },
  //...
}
```

6. Repeat this for each font in your app, adding another entry to your theme
   under `baselineCrop`.

### Usage

```html
<Box height="10px" width="100%" bg="green.800" />
<Text baselineCrop>Mark Dalgleish: Meme Lord</Text>
<Box height="10px" width="100%" bg="green.800" />
```

## Props

Text composes the [Box](/box) component, so you can pass all `Box` style props.
